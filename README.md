# Browser-Automation-Extension

> Browser Automation Extension is a Chromium based side panel automation tool that records browser actions like clicks, typing, and navigation, then replays them as reusable automations. It also lets users edit, loop, import, and export recorded workflows directly from browser storage.

![Browser-Automation-Extension](repo-card.png)
Built by [Naadir](https://github.com/Naadir-Dev-Portfolio)

---

## Overview

Records real user interactions in the browser and converts them into structured, replayable automation steps. Eliminates manual repetition for workflows like form filling, navigation, and testing. Provides a deterministic replay engine with timing control and fallback element selection.

---

## Features

- Records click, input, and navigation events directly from live browser sessions  
- Builds robust multi-strategy selectors (CSS, XPath, text, data attributes) for reliable replay  
- Replays workflows with timing control, delay scaling, and navigation synchronization  
- Supports loop execution with configurable delay and iteration count  
- Allows editing, importing, exporting, and persistent storage of automations via browser storage
